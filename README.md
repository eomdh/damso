## 💡 구현 기능
- 회원가입 (bcrypt로 패스워드 암호화)
- 로그인, 로그아웃 처리(passport 이용, cookie/session 처리)
- 프로필 이미지 등록(multer), 대화명 설정
- 게시글 등록, 수정, 삭제
- 댓글 등록, 수정, 삭제
- 인피니티 스크롤링
- 좋아요
- 이미지 업로드(multer)
- 이미지 캐루셀 구현(react-slick)
